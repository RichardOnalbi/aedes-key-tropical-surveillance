# Illustrated Identification Key for Emerging Non-native *Aedes* of Tropical Medicine Relevance

This repository contains an illustrated interactive key developed to support the rapid recognition of emerging non-native *Aedes* mosquitoes of surveillance relevance in tropical settings.

The site includes:

- a dichotomous identification key;
- a visual atlas for rapid comparison;
- Spanish and English versions;
- diagnostic image panels for key taxonomic characters;
- operational emphasis for public health technicians and entomological surveillance.

## Main files

- `index.html` — Spanish version
- `index-en.html` — English version

## Purpose

The key is intended as a rapid-use surveillance aid. It does not replace specialist taxonomic review or molecular confirmation, but helps support early suspicion, specimen triage, and field-to-laboratory decision-making.

## How to use

Open `index.html` or `index-en.html` in a web browser, or access the GitHub Pages version once the repository is published.

## Project context

This tool was developed as part of a manuscript on *Aedes albopictus* as a model for anticipatory surveillance of emerging non-native *Aedes* of tropical medicine relevance.
